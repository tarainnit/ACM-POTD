## 📌 Challenge Goal

* Solve **1 problem every day**
* Stay consistent for 30 days
* Build problem-solving skills step by step

## 📂 Structure

* `Tripti_Date_Day-1.md`
* `Tripti_Date_Day-2.md`
* ...

Each file includes:

* Problem link
* Approach
* Code
* Screenshot of output
